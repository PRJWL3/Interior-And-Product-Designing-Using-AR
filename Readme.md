🏠 Interior and Product Design Using AR

📌 Overview

This project is an Augmented Reality (AR) based mobile application that allows users to visualize and place virtual furniture and products in real-world environments using a smartphone camera. It helps users make better interior design decisions by providing a real-time interactive experience.

---

🚀 Features

* 📱 Real-time AR visualization
* 🛋️ Virtual furniture/product placement
* 🔄 Move, rotate, and scale objects
* 📏 Accurate surface detection using ARCore
* 🎯 User-friendly interface

---

🏗️ Architecture

* Camera captures real-world environment
* ARCore detects planes (floor, walls, surfaces)
* Unity processes input using AR Foundation
* Virtual objects are placed and rendered in real time

---

🧰 Technology Stack

* Frontend: Unity (UI & 3D Rendering)
* Backend: C#
* Frameworks: AR Foundation, ARCore
* Platform: Android
* 3D Assets: FBX models (Blender/Mixamo)

Uses ARCore to detect surfaces like floors and walls using camera input.

Allows users to place, move, rotate, and scale virtual objects in the real world.
